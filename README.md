# Troubleshooting Guide (EN / PT-BR)

## 🇺🇸 Overview (English)

A practical, bilingual troubleshooting guide covering common technical issues and their step-by-step solutions. Every topic is documented in both **English** and **Brazilian Portuguese**, making the guide useful for international teams and Brazilian developers alike.

Each guide follows the same philosophy: start from the most common cause, work toward the least common, and explain the *why* behind each fix — not just the quick command.

### Repository files

| File / Folder | Purpose |
|---|---|
| `README.md` | This file. Project overview, repository structure, and usage instructions in English and Portuguese. |
| `LICENSE.md` | The full text of the MIT License under which this project is distributed. |
| `en/` | Troubleshooting guides written in English (one Markdown file per topic). |
| `pt-br/` | The same guides translated into Brazilian Portuguese, mirroring the structure of `en/`. |

> Note: the `en/` and `pt-br/` folders are populated as new guides are added. Each guide in `en/` must have a matching file with the same name in `pt-br/`.

### How to use

1. Open the folder matching your language (`en/` or `pt-br/`).
2. Find the topic that matches the problem you are facing.
3. Follow the diagnostic steps in order — each guide is sorted from the most likely cause to the least likely one.

### Contributing

Contributions are welcome! Please keep both language versions in sync: if you update a guide in `en/`, update the matching file in `pt-br/` — or open an issue flagging that a translation is pending.

## 🇧🇷 Visão Geral (Português)

Um guia de troubleshooting prático e bilíngue, cobrindo problemas técnicos comuns e suas soluções passo a passo. Cada tópico é documentado em **inglês** e em **português do Brasil**, tornando o guia útil tanto para times internacionais quanto para desenvolvedores brasileiros.

Todos os guias seguem a mesma filosofia: começar pela causa mais comum, avançar até a menos comum, e explicar o *porquê* de cada correção — não apenas o comando rápido.

### Arquivos do repositório

| Arquivo / Pasta | Finalidade |
|---|---|
| `README.md` | Este arquivo. Visão geral do projeto, estrutura do repositório e instruções de uso em inglês e português. |
| `LICENSE.md` | O texto completo da Licença MIT sob a qual este projeto é distribuído. |
| `en/` | Guias de troubleshooting escritos em inglês (um arquivo Markdown por tópico). |
| `pt-br/` | Os mesmos guias traduzidos para o português do Brasil, espelhando a estrutura de `en/`. |

> Observação: as pastas `en/` e `pt-br/` são preenchidas conforme novos guias são adicionados. Cada guia em `en/` deve ter um arquivo correspondente com o mesmo nome em `pt-br/`.

### Como usar

1. Abra a pasta do seu idioma (`en/` ou `pt-br/`).
2. Encontre o tópico que corresponde ao problema que você está enfrentando.
3. Siga os passos de diagnóstico em ordem — cada guia está ordenado da causa mais provável para a menos provável.

### Contribuindo
